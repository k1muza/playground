# **App Name**: LearnVerse

## Core Features:

- Course Listing: Display a list of courses with titles, descriptions, and topic badges.
- Problem Sets: Show a list of practice problems, tagged with difficulty and relevant topics.
- Article Library: Provide a catalog of articles, each with a title, excerpt, and tags.
- Search Functionality: Implement a search bar that filters courses, problems, and articles based on keywords.
- Course Detail Pages: Create detailed pages for each course, including a sticky sidebar for navigation, lesson content, and dynamic table of contents generated with AI tool

## Style Guidelines:

- Primary color: Midnight Blue (#191970) to evoke a sense of depth and knowledge.
- Background color: Light Gray (#F0F0F0) for a clean and neutral backdrop.
- Accent color: Sky Blue (#87CEEB) to provide highlights and interactive elements with a calm, engaging tone.
- Body and headline font: 'Inter' sans-serif for a modern and neutral appearance suitable for both headlines and body text.
- Code font: 'Source Code Pro' monospaced, specifically for displaying code snippets within articles and lessons.
- Use clean, outline-style icons from lucide-react for a consistent and modern feel.
- Implement a responsive layout using Tailwind CSS grid and flexbox to ensure optimal viewing on various devices.
